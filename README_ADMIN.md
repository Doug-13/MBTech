# MB Tech - Página administrativa

Acesse a administração pela rota:

```txt
/admin
```

Login inicial:

```txt
E-mail: admin@mbtech.com.br
Senha: 123456
```

Antes de usar, execute no Supabase o arquivo:

```txt
mb-tech-seed-platform-admin.sql
```

Na página administrativa você consegue:

- criar empresas clientes;
- editar status e dados básicos das empresas;
- selecionar uma empresa;
- criar usuários da empresa para acessar o painel do cliente;
- ativar/inativar usuários;
- alterar senha do usuário.
