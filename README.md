# Front MB Tech conectado ao backend

Arquivos principais:

- `src/App.jsx`: tela sem mocks, consumindo API real.
- `src/services/api.js`: cliente HTTP com token JWT salvo no localStorage.
- `.env.example`: exemplo da URL da API.

## Como aplicar no seu front Vite

1. Copie `src/App.jsx` para o seu projeto.
2. Copie `src/services/api.js` para `src/services/api.js`.
3. Crie `.env.local` no front:

```env
VITE_API_URL="http://localhost:3001/api"
```

4. Reinicie o front:

```bash
npm run dev
```
